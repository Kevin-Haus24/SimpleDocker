# SimpleDocker

Учебный проект по Docker, в котором последовательно собраны задания от базовой работы с образами до собственного образа, проверки через Dockle и запуска через docker-compose.

## Что внутри

- Работа с готовым образом `nginx`, контейнерами и портами.
- Настройка `nginx` для отдачи статуса и проксирования запросов.
- Мини-сервер на `C` и `FastCGI`, запускаемый через `spawn-fcgi`.
- Собственный Docker-образ с `nginx` и сборкой мини-сервера.
- Доработка образа под требования `dockle`.
- Финальный запуск через `docker-compose.yml`.

## Структура

- `src/report.md` — подробный отчёт по всем частям проекта со скриншотами.
- `src/Task 3/` — мини-сервер на `C` и конфигурация `nginx`.
- `src/Task 4/` — Dockerfile для сборки образа с FastCGI-сервером и `nginx`.
- `src/Task 5/` — улучшенная версия образа с учётом проверок `dockle`.
- `src/Task 6/` — финальная конфигурация `docker-compose.yml`.

## Результат

Полное описание выполнения - `src/report.md`.

Финальный сценарий запуска - `src/Task 6/docker-compose.yml`.
